傑克玩樂園 CHUNITHM 線下比賽

Event Information

System time :

一、活動時間

2024-08-10 (六) 14:00

二、活動地點

高雄夢時代 B2 綠區 傑克玩樂園

三、活動流程

1. 賽制採單淘汰制，由玩家抽選籤號決定賽程。

2. 比賽歌曲選擇皆為 Master 譜面隨機曲，惟只計前三首遊戲紀錄。

3. 進行每首歌曲前，於籤箱抽籤決定獲勝條件，籤箱內容分別為：JUSTICE CRITICAL 比低 / JUSTICE 比高 / ATTACK 比高 / Max Combo 比低

4. 每輪比賽兩位玩家於開始前先進行猜拳，勝者於 Track 1 及 Track 3 抽籤及抽選譜面，負者於 Track 2 抽籤及抽選譜面，賽制為三戰兩勝。

四、活動規則

1. 活動無需報名費，玩家請自行投幣。比賽期間需將 Combo 數值顯示關閉。

2. 倘若其中一位玩家分數未達 90 萬分，則該位玩家判負；若兩位玩家若分數皆未達 90 萬分，以成績最接近 90 萬分者為勝。

3. 若遇比賽項目其雙方紀錄皆相同時，以成績最接近 90 萬分者為勝。

4. 工作人員將視玩家情況，於抽選難度過高的歌曲時重新抽選譜面。

5. 請於工作人員完成紀錄每首歌曲結算畫面後再繼續遊戲。

6. 比賽最終規則會於賽前再次公佈，如有任何異議，請當場立即反應。
