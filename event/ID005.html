Jack｜LINE 點數獲得社群活動 (2021-08)

Event Information

Last updated : 2021年08月20日 17:26


一、活動期間

2021-08-20 (五) 17:00~19:59

二、活動位置

LINE 點數獲得社群 -> 活動記事本

三、活動流程

於活動期間內前往活動記事本留言即有機會獲得好禮。

四、活動規則

1. 活動期間單一成員最多限留言一次，多次留言將取消獲獎資格。

2. 留言格式請由 100 開始依序報數 (例：100、101、102....以此類推)。

3. 獲獎時遇重複報號情形以首位報號者計算之，其餘皆不計入。

4. 本活動獎項為機會中獎，贈獎抽選結果將於活動結束後公告於社群聊天室。

5. 本活動限時 3 小時，請務必於活動時間內完成留言，若未能於期限內完成活動流程，則視為放棄。

五、活動獎項

家樂福錢包 50 元儲值金 (共6名)

社群活動領獎方式

1. 得獎者請於活動結束後點選此身分確認連結，將 userID 後 5 碼傳送至活動記事本留言處並至帳號ID「event_jack」回覆，待身分確認無誤後，將儘速派發尚未發放之獎項。

2. 得獎者需於 2021-08-23 前完成身分確認，逾期視為得獎者自動放棄中獎資格及獎項，得獎者不得異議。

4. 若發生無法正常運作的情形，請將 LINE 應用程式升級為最新版本。

5. 若因個人因素導致活動無法順利完成或獲得之贈品遺失，將不負擔任何責任或損害賠償。
